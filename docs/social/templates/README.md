# Instagram Templates

Use these templates as starting points, not rigid designs. Each template names the content job, best-fit metric, slide or frame structure, visual system, and safety notes.

## Template Index

- `educational-carousel.md`: best for saves, shares, and beginner charting education.
- `reach-first-reframe.md`: best for reach, shares, follows, and profile visits.
- `privacy-trust-post.md`: best for trust, profile visits, follows, and saves.
- `community-prompt.md`: best for comments, audience language, and account warmth.
- `product-light-demo.md`: best for profile visits, link taps, and app understanding.

## How To Use

1. Choose a draft from `../INSTAGRAM_DRAFT_QUEUE.md`.
2. Choose one template that matches the content role and target metric.
3. Keep one primary variable from `../INSTAGRAM_DECISION_RUBRIC.md`.
4. Apply `well-within-social-creative-direction` before asset creation. **Design or lock the hook slide first**, then align later slides to that visual system; prefer **4–5 slides** total when the story fits.
5. Treat these files as content structures, not visual designs. Default to the journal-led north star; use app-led visuals only when the content job needs real product proof.

## Template Rules

- Do not copy competitor posts or layouts.
- Do not use predictive, diagnostic, treatment, effectiveness, or pregnancy outcome claims.
- Do not invent or approximate the Well Within logo.
- Keep text readable on mobile.
- Prefer one idea per slide.
- Avoid boxed carousel/card layouts as the default look.
- If a lower-capability model is running the post, choose a template and avoid improvising complex visual direction.
